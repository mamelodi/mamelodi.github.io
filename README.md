# The Mamelodi Initiative Website

This repository contains the Jekyll source code for The Mamelodi Initiative website, an educational non-profit in South Africa.

## Development

This is a static site built with [Jekyll](https://jekyllrb.com/) and hosted on GitHub Pages. It uses the [Minimal Mistakes](https://github.com/mmistakes/minimal-mistakes) remote theme.

To run locally:
```bash
bundle install
bundle exec jekyll serve --livereload
```
